# furniture_factory_project
13. Мебельная фабрика
На мебельной фабрике выполняется производство и изготовление корпусной и мягкой мебели. Производство разделено на несколько цехов, в каждом из которых работает некоторое количество рабочих. У каждого цеха есть начальник, контролирующий процесс выполнения заказов. Для выполнения заказов может потребоваться работа в нескольких цехах. Информационная система предназначена для администратора фабрики.
